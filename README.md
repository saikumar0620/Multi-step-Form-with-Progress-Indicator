# Multi-Step Form

This is a simple, vanilla HTML/CSS/JS project that builds a multi-step registration form with a progress bar. I built this to practice DOM manipulation, event handling, and form validation.

## Demo

- **Live demo:** [https://saikumar0620.github.io/Multi-step-Form-with-Progress-Indicator/](https://saikumar0620.github.io/Multi-step-Form-with-Progress-Indicator/)

## Features

- **Progress Tracker:** A visual bar and step circles that update as you navigate through the form.
- **Form Validation:** Checks if inputs (like email, phone, and credit card) are entered correctly before letting you proceed to the next step.
- **Auto-Formatting:** Automatically adds spaces to the credit card number and slashes to the expiry date while you type.
- **Review Step:** Shows a summary of all the information you entered (masking the card number) before the final submission.
- **Success Screen:** Displays a clean confirmation message when you finish, with an option to reset the form.

## How to Run

Since this is built with basic web technologies, there's no need for complex setup, servers, or build tools.

1. Clone or download the folder.
2. Just double-click the `index.html` file to open it directly in your web browser.
3. Try filling out the form to see the validation and animations in action!

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS Variables)
- Vanilla JavaScript
- FontAwesome (for the icons)
