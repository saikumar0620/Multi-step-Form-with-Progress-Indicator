# Progress Step

This project showcases a step progress indicator. It visually represents the progress of a task or procedure in steps.

## Deployed Link
You can view the live demo of the application [here](https://saikumar0620.github.io/progress-step/).

## Features
- Visual representation of steps
- Interactive design
- Responsive layout

## Installation
Run the following command to install dependencies:
```
npm install
```

## Usage
To run the application locally, execute:
```
npm start
```
